# Social Media Dashboard

A modern social media management platform built with React, TypeScript and Vite.

## Features

- Multi-platform social media management
- Content scheduling
- Analytics dashboard
- AI content assistant
- Team collaboration
- Unified inbox

## Tech Stack

### Frontend
- React
- TypeScript
- Vite
- React Router

### Planned Backend
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
